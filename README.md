# sdu-football-ai
AI analysis # SDU Football League AI Analysis ⚽

## Overview
This project analyzes the SDU Football League (SFL) using real match data.

## Data
- Match results (CSV file)
- Screenshots from Telegram channel

## Features
- Team performance analysis
- Match result tracking
- Basic insights about league standings

## Example Insights
- Which team wins most often
- Goal statistics
- Team rankings

## Tools Used
- Excel / CSV
- ChatGPT (AI analysis)

## Project Goal
To demonstrate how AI can be used to analyze real sports data.
